# Cyclistic Bike-Share Analysis

## Project Overview
This project analyzes how annual members and casual riders use Cyclistic bikes differently.

Cyclistic is a fictional company used in the Google Data Analytics case study.

## Business Task
Identify key differences in behavior between casual riders and annual members to inform marketing strategies.

## Data Source
Data sourced from Divvy trip data (public dataset).

## Tools Used
- Google BigQuery (SQL)
- Data Cleaning & Transformation
- Data Visualization (Google Sheets / Presentation)

## Key Findings
- Casual riders:
  - Ride more on weekends (+49%)
  - Take longer trips (1356 sec vs 740 sec)
  - Ride more in warmer months (+368%)

- Annual members:
  - Ride more on weekdays (+31%)
  - Take shorter trips
  - Show consistent usage year-round

- Rideable type:
  - Minimal difference (<3%) between groups

## Business Recommendations
- Target casual riders with:
  - Weekend promotions
  - Seasonal campaigns (spring/summer)
- Emphasize membership benefits for frequent commuters
